## How It Works

```mermaid
flowchart LR
    USER["User types question"] --> AI["AI processes intent"]
    AI --> SQL["SQL generated"]
    SQL --> DATA["Data retrieved"]
    DATA --> VIZ["Visualization selected"]
    VIZ --> STREAM["Response streamed"]

    style USER fill:#0D9373,color:#fff
    style STREAM fill:#07C983,color:#fff
```

<Steps>
  <Step title="Ask in Natural Language">
    Type your question exactly as you'd ask a colleague: *"What were our top products last quarter?"*
  </Step>
  <Step title="AI Understands Intent">
    Superatom classifies your intent, extracts entities, and applies tribal knowledge for context.
  </Step>
  <Step title="Query Generated & Executed">
    SQL is automatically generated and run against your connected data sources.
  </Step>
  <Step title="Visualization Selected">
    The AI selects the optimal visualization from 16+ component types based on your data.
  </Step>
  <Step title="Response Streamed">
    Results appear in real-time with interactive charts, follow-up suggestions, and the underlying query.
  </Step>
</Steps>

### Real-Time Streaming

Responses appear as they're generated—no waiting for the complete answer:

- **Text streams** character by character
- **Charts build** progressively
- **Follow-ups appear** after main response

### Interactive Visualizations

Every response includes the optimal visualization:

| Data Type | Auto-Selected Component |
|-----------|------------------------|
| Single metric | KPI Card |
| Categorical comparison | Bar Chart |
| Time series | Line Chart |
| Proportions | Pie/Donut Chart |
| Detailed records | Data Table |
| Geographic | Map Chart |

### Transparency

Every query shows the underlying SQL:

```sql
-- Generated for: "Top 10 products by revenue this quarter"
SELECT
    product_name,
    SUM(revenue) as total_revenue
FROM sales
WHERE sale_date >= '2026-01-01'
GROUP BY product_name
ORDER BY total_revenue DESC
LIMIT 10
```

Users can:
- **Verify** the logic is correct
- **Learn** how queries work
- **Modify** if needed (advanced users)

### Follow-Up Suggestions

After each response, the AI suggests relevant follow-up questions:

- *"Show breakdown by region"*
- *"Compare to same period last year"*
- *"What's driving this trend?"*

## Commands

Power users can use commands for quick actions:

| Command | Action |
|---------|--------|
| `/clear` | Clear conversation history |
| `/help` | Show available commands |
| `/bookmark` | Save current visualization |
| `/export` | Export data as CSV |

---

## Bookmarks

Save useful insights for quick access:

1. Find a valuable visualization
2. Click the **bookmark icon** or type `/bookmark`
3. Add a name and optional notes
4. Access anytime from the Bookmarks panel

Bookmarks preserve:
- The exact query
- The visualization configuration
- Point-in-time or live data (configurable)

---

## Conversation History

All conversations are saved with full context:

- **Thread grouping** — Related questions grouped together
- **Searchable** — Find past queries by keyword
- **Audit trail** — Who asked what, when
- **Resumable** — Continue previous analysis sessions

---

## Access Control

The Chat Agent respects role-based permissions:

| Role | Capabilities |
|------|-------------|
| Viewer | Ask questions, view responses |
| Analyst | All viewer + save bookmarks, export |
| Admin | All analyst + configure data sources |

Data access is filtered based on user permissions—users only see data they're authorized to access.
