## Dashboard Basics

Dashboards combine multiple visualizations into a single view.

## Creating a Dashboard

<Steps>
  <Step title="Open Dashboards">
    Click Dashboards in the sidebar
  </Step>
  <Step title="Create New">
    Click + Create Dashboard
  </Step>
  <Step title="Add Components">
    - Type questions
    - Add from bookmarks
    - Use component library
  </Step>
  <Step title="Arrange Layout">
    Drag and resize components
  </Step>
  <Step title="Save & Share">
    Name and set permissions
  </Step>
</Steps>

## Component Types

| Type | Use Case |
|------|----------|
| KPI Card | Single metrics |
| Bar Chart | Comparisons |
| Line Chart | Trends |
| Table | Detailed data |
| Filter | User interaction |

## Design Tips

<AccordionGroup>
  <Accordion title="Keep it focused">

  <Accordion title="Prioritize position">

  </Accordion>
  <Accordion title="Use filters">

  </Accordion>
</AccordionGroup>
