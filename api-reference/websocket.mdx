## Connection

Connect to the WebSocket endpoint:

```javascript
const ws = new WebSocket('wss://ws.superatom.ai/v1');

ws.onopen = () => {
  ws.send(JSON.stringify({
    type: 'auth',
    token: 'YOUR_API_KEY'
  }));
};
```

## Message Types

### Query

```json
{
  "type": "query",
  "id": "msg_123",
  "content": "Show me sales by region"
}
```

### Response (Streamed)

```json
{
  "type": "chunk",
  "id": "msg_123",
  "content": "Here are the sales..."
}
```

### Complete

```json
{
  "type": "complete",
  "id": "msg_123",
  "visualization": { ... }
}
```

## Auto-Reconnection

Handle disconnections gracefully:

```javascript
ws.onclose = () => {
  setTimeout(reconnect, 1000 * Math.pow(2, attempts));
};
```

## Heartbeat

Send periodic pings to keep connection alive:

```json
{ "type": "ping" }
```
