## Overview

The Superatom API provides programmatic access to the platform's capabilities.

## Base URL

```
https://api.superatom.ai/v1
```

## Authentication

All API requests require authentication using an API key:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
     https://api.superatom.ai/v1/query
```

## Rate Limits

| Plan | Requests/minute |
|------|-----------------|
| Starter | 60 |
| Professional | 300 |
| Enterprise | Custom |

## Response Format

All responses are JSON:

```json
{
  "success": true,
  "data": { ... },
  "meta": {
    "request_id": "req_abc123",
    "duration_ms": 234
  }
}
```

## Error Handling

```json
{
  "success": false,
  "error": {
    "code": "INVALID_REQUEST",
    "message": "Description of the error"
  }
}
```
