Superatom automatically generates mobile applications from your data model—giving field workers, executives, and anyone on the go instant access to insights and actions.

  ![Superatom Mobile App](/images/mobile-app.png)

---

## The Accessibility Problem

Traditional approach to mobile data access:
- **Months** of development time
- **$50K-200K** development cost
- **Ongoing maintenance** burden
- **Separate codebase** to manage

**Superatom approach:** Mobile apps are automatically generated from your data model. Zero development required.

---

## How It Works

<MobileAppsFlowDiagram client:load />

1. **Connect Data Sources**

   Your data is already connected to Superatom

  1. **Define Mobile Views**

   Select which dashboards, reports, and actions are mobile-enabled

  1. **Configure Roles**

   Set permissions for different user types

  1. **Deploy**

   Apps are instantly available on iOS and Android

## Role-Based Access

Different users see different things:

| Role | Mobile Access |
|------|--------------|
| **Executive** | High-level KPIs, trend summaries |
| **Manager** | Team metrics, approval workflows |
| **Field Worker** | Task-specific views, action buttons |
| **Analyst** | Full query access, detailed data |

---

## Offline Capabilities

Work without internet connection:
- **Cached data** — Recent queries available offline
- **Queued actions** — Submit actions when reconnected
- **Sync indicator** — Know when data was last updated

---

## Push Notifications

Stay informed with smart alerts:

| Notification Type | Example |
|------------------|---------|
| **Threshold Alert** | "Inventory below safety stock at Warehouse A" |
| **Scheduled Report** | "Your weekly summary is ready" |
| **Action Required** | "Transfer request needs approval" |
| **Anomaly Detected** | "Revenue 15% below forecast" |

Configure notification preferences:
- Which alerts to receive
- Quiet hours
- Frequency limits
