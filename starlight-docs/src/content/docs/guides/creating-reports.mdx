## Report Types

| Type | Description |
|------|-------------|
| Scheduled | Run on a recurring basis |
| Triggered | Run when conditions are met |
| Ad-hoc | Generated on demand |

## Creating a Scheduled Report

1. **Open Reports**

   Navigate to Reports / AI Analyst

  1. **Create Report**

   Describe what you want to analyze

  1. **Configure Schedule**

   Set frequency and timing

  1. **Set Recipients**

   Choose who receives the report

  1. **Activate**

   Turn on the schedule

## Schedule Options

| Frequency | Examples |
|-----------|----------|
| Daily | Every day at 7 AM |
| Weekly | Every Monday morning |
| Monthly | First of the month |
| Quarterly | Start of each quarter |

## Delivery Methods
