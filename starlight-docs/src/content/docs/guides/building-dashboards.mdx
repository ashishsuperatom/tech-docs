## Dashboard Basics

Dashboards combine multiple visualizations into a single view.

## Creating a Dashboard

1. **Open Dashboards**

   Click Dashboards in the sidebar

  1. **Create New**

   Click + Create Dashboard

  1. **Add Components**

   - Type questions
    - Add from bookmarks
    - Use component library

  1. **Arrange Layout**

   Drag and resize components

  1. **Save & Share**

   Name and set permissions

## Component Types

| Type | Use Case |
|------|----------|
| KPI Card | Single metrics |
| Bar Chart | Comparisons |
| Line Chart | Trends |
| Table | Detailed data |
| Filter | User interaction |
