## Connection Steps

1. **Navigate to Data Sources**

   Knowledge Base → Data Sources

  1. **Select Type**

   Choose your database type

  1. **Enter Details**

   Host, port, database, credentials

  1. **Test Connection**

   Verify connectivity

  1. **Run Analysis**

   Superatom analyzes your schema

## Connection Details

### PostgreSQL

```
Host: your-host.com
Port: 5432
Database: your_database
Username: your_user
Password: ********
SSL: Required
```

### BigQuery

```
Project ID: your-project
Dataset: your_dataset
Service Account: credentials.json
```

## Best Practices

- Use read-only credentials
- Enable SSL/TLS
- Whitelist Superatom IPs
- Test before production use
