## AI Pipeline

```mermaid
flowchart TB
    INPUT["User Question"] --> CLASSIFY["Intent Classification"]
    CLASSIFY --> CONTEXT["Context Enrichment"]
    CONTEXT --> GENERATE["Query Generation"]
    GENERATE --> EXECUTE["Execution"]
    EXECUTE --> ANALYZE["Result Analysis"]
    ANALYZE --> VISUALIZE["Visualization Selection"]
    VISUALIZE --> OUTPUT["Streamed Response"]
```

## Caching Strategy

- **Query cache:** 5-minute TTL for repeated queries
- **Embedding cache:** Vector representations persist
- **Component cache:** Frequently-used configs cached
